夜莺与玫瑰
奥斯卡·王尔德


◆ 快乐王子

>> 一只小燕子飞过这座城市的夜空。他的朋友们在六个星期前就飞往埃及去了，但他留了下来，因为他爱上了那株最美的芦苇。春天的时候，燕子在沿河追逐一只黄色大蛾子时遇见了芦苇。她苗条的腰肢深深地打动了他，他停下来和她搭话。 
    “我可以爱你吗？”燕子说话喜欢直截了当。芦苇对他低低地鞠了一躬，于是燕子绕着芦苇飞了一圈又一圈，用翅尖点着水，水面上泛起一圈又一圈的银色涟漪，这是燕子求爱的方式。就这样整个夏天过去了。 

>> 快乐王子不再美丽，那么他也就没有用了。

>>     “把这座城里最珍贵的两样东西给我带来。”上帝对一位天使说。于是天使把那颗铅心和那只死鸟带了回来。 
    “你选得对！”上帝说，“这只小鸟将在天堂花园里永远歌唱，而快乐王子将在我的黄金城里永远赞美我。”

◆ 少年国王 THE YOUNG KING

>> “打仗的时候，”织工说，“强的那一方让弱的成为奴隶；不打仗的时候，有钱人让穷人当奴隶。

>> 潜水的人最后一次浮上来了，这一次他采回的珍珠比霍尔木兹所有的珍珠还要精致，浑圆如满月，雪白如晨星。但他的脸上已经毫无血色了，他倒在甲板上，鲜血从耳朵和鼻孔喷出。身子轻轻颤了几下，再也不动了。黑人们耸了耸肩，把尸体扔进海里。

>> 悲哀的纺车、痛苦的双手织成了这件长袍。红宝石里流的是鲜血，珍珠的内核是死亡。”

>> 人群中走来一个人，面目凄苦地说：“陛下，您有所不知，有富人的奢华，才有穷人的生计。您的铺张浪费让我们能赚到口吃的。给主人家干活儿是辛苦，但不能给主人干活儿反而更苦。您以为我们是靠乌鸦喂饱的吗？您有法子改变现状吗？您能要求买家‘必须以这个价格购买’，能要求卖家‘必须以这个价格卖出’吗？我想是不能。所以还是回宫殿去吧，穿上您的王室华服。我们的痛苦，关您什么事呢？”

>> 这世界的担子如此沉重，哪是一个人能负担得起的？这世界的痛苦如此深刻，哪是一个心脏能忍受得了的？


◆ 西班牙公主的生日 THE BIRTHDAY OF THE INFANTA

>> 小矮人听说他又能在公主面前跳一次舞，还是公主亲自吩咐的，喜不自胜，转眼就跑进了花园，疯疯癫癫地吻着一朵白玫瑰，姿势别扭而丑陋，但他实在太高兴了。

>> 相貌平平就是他的错，不能因为一个人无药可救而去欣赏他。

>> 王宫里的空气又沉又闷，但森林里吹着清风，日光挥动着金手拂开颤抖的树叶。森林里也有美丽的花朵，也许不如园子里的花那么惊艳，但花香却格外甜蜜浓郁。早春时节的风信子在清凉的幽谷中、在茂密的草丘上翻滚着紫色花浪；丛丛黄色的樱草依着橡树嶙峋的老根而生；还有鲜亮的白屈菜、蓝色的虎尾草、紫色和金色的鸢尾花。榛树上长了灰色柔荑花。毛地黄因花房的重量而沉沉下坠，布满花斑的花房常常吸引蜜蜂流连。栗树有它白星一般的尖尖小花，山楂有它冷月似的浅色花丛。

>> 当真相终于浮出水面时，小矮人绝望地嘶吼，倒在地上抽泣起来。原来他就是那个奇形怪状、驼背跛足的丑八怪，丑得让人不愿多看一眼。他就是那个怪物，所有孩子都在嘲笑他，而他一直以为深爱自己的小公主，也只是在嘲笑他的丑陋、拿他一瘸一拐的腿脚寻开心罢了。为什么他们不让他待在森林呢？那里没有镜子能告诉他，他有多么面目可憎。为什么父亲不杀了他，而把他当小丑卖了呢？滚烫的泪水顺着脸颊流下，他把那朵白玫瑰撕得粉碎。那个蜷缩在地上的怪物也这么做了，把撕碎了的花瓣朝天上乱扔。它趴在地上，当他看着它时，它也看回去，两张憔悴的面孔同样因痛苦扭曲。他不敢再看了，便捂着眼睛向远处爬开。像一只受了伤的动物爬进阴影中，躺在那儿苦苦呻吟。

>> “我美丽的公主，您的小矮人再也不能跳舞了。很遗憾，他长得这么丑，本来有机会能逗国王一乐呢。”“但他为什么不能跳舞了呀？”公主大笑着问。“因为他的心碎了。”大臣答道。小公主皱起眉头，那两片玫瑰叶子一样秀丽的小嘴唇噘了起来。“以后让没有心的人进宫陪我玩吧！”她大声说完，就跑进花园去了。


◆ 渔夫和灵魂 THE FISHERMAN AND HISSOUL

>> 灵魂是人最高贵的一部分，是上帝所赐予的，我们必须以崇高之心看待它啊。没有什么比人的灵魂更珍贵的了，世间没有一样东西可以和它相比。它的价值相当于世上所有的金子，比所有国王的宝石加起来还要珍贵。所以，我的孩子，你就别再想这事了，这是不能被原谅的罪。至于人鱼，他们早已堕落，所有跟随他们的人也将要堕落。他们像田野里的怪兽一样，不分善恶，主的牺牲并不是为了他们。

>> “走吧！走吧！”神父哀叹道，“你的爱人已经迷失了，你将和她一起堕落。”

>> 商贩们大笑起来，说：“你的灵魂对我们又有何用呢？还不如半枚银币值钱。把你的身体卖给我们当奴隶吧，我们给你穿上紫色衣裳，手指套上戒指，然后送到王后陛下那儿当小丑。别再说什么灵魂了，它什么也算不上，一点儿用也没有。”年轻的渔夫自言自语：“多蹊跷啊！神父跟我说灵魂价值相当于这世上所有的金子，但商人又说它值不过半枚银币。”

>> 人们所谓的影子，其实并不是肉体的影子，而是灵魂的肉体。你站在沙滩上，背朝月亮，沿着双脚把地上的影子割开，那就是你的灵魂了。

>> 窄窄的街巷里，无数只美丽的纸灯笼像花蝴蝶一样翩翩起舞。风吹过屋顶时，那灯笼浮浮沉沉，仿佛五颜六色的肥皂泡。

>> 因为世界上没有送不走的痛苦，也没有迎不来的快乐。

>> 爱情比智慧伟大，比财宝珍贵，比少女的玉足更美。烈火无法摧毁它，海水不能淹没它。我在黎明呼唤你，可你没有回答。月亮都听见我叫你的名字，可你还是不理我。我扔下你走向邪恶，却颠沛流离，自食其果。但你的爱永远陪伴着我，爱情永远强大，无可战胜，即使我目睹了所有善与恶。现在，你死了，就让我和你一起走吧。
